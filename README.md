# ZinvainOS Backend

Enterprise Operating System Backend API

## Tech Stack
- Node.js
- Express.js
- PostgreSQL (Supabase)
- JWT Authentication

## Installation

```bash
# Install dependencies
npm install

# Copy environment file
cp .env.example .env

# Update .env with your credentials

# Run database migrations
npm run migrate

# Seed initial data (CEO account)
npm run seed

# Start development server
npm run dev
